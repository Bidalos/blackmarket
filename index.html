Blackmarket

blackmarket is a fan made application for the free to play MMORPG game Path of Exile

View project on GitHub

Blackmarket is a Path of Exile items search tool that allows you to search what you want in easiest possible way. As easy just typing in "90life 60res gloves" in the search field and hit 'enter'. This will search for gloves with at least 90 life, at least 60 total elemental resistances, with buyout and online only, sorted cheapest first.

Requirements

You'll need to have the Java Runtime Environment (JRE) installed. You can download the latest version of Java here. JRE is just like .NET Framework which is also a runtime environment, only easier to install and better :P. Blackmarket is tested with Java 8, but it should work with Java 7 as well.

How to run

Double click the blackmarket-.jar. As simple as that.

Basic Usage

  1. Type in what you wanna search, for example: 500pdps 2hsword 5L6s
  2. Click Search and wait for the results to show up.
  3. You click on any of the items to open up the Item View Window.
  4. Once you've decided to buy something, click WTB button. This will copy a generated WTB message to your clipboard.
  5. Switch to in-game, open the chat window and press CTRL+V.
  6. Be friendly and happy trading!

Note on defaults

  • League - Set to Warbands by default
  • With Buyout Only - Set to true by default. To disable this, set buyoutOnly=false, buyoutOnlyCurrency= You can change this via blackmarket.properties file which is found in your home directory/.blackmarket. E.g. C:\Users<YourUserName>.blackmarket.

Advance Usage

  1. The "Blackmarket Language" is designed to be customizable and extensible via blackmarket.language file. The guide on editing this file has not yet materialized but here is an example:
    • cod = name=Cloak of Defiance
    • veil = name=Veil of the Night
