Blackmarket

The exiles' ultimate way to find the best items in wraeclast.

This contains the blackmarket runnable jar, note that you'll need to install Java 8 update 60 first.

Blackmarket is a Path of Exile items search tool that allows you to search what you want in easiest possible way.

An important note on Open Source

Blackmarket is 100% free and fully open source under GPLv2. In a nutshell, this means that Blackmarket will always be free, and anyone can take the source code and modify it for improvements. However, this will also require that all the released modified versions be free software as well.

Read more about open source here. The GNU GPLv2 FAQ page is also a good read.

Download

Download latest version now from the Releases page. You'll see 3 downloads there, you'll want the .jar file. The other two contains the full source code of blackmarket for the given release.

Requirements

You'll need to have the Java Runtime Environment (JRE) installed. You can download the latest version of Java here. JRE is just like .NET Framework which is also a runtime environment, only easier to install and better :P. Starting v0.4 Blackmarket requires Java 8 update 60

How to run

Double click the blackmarket-<version>.jar. As simple as that.
